# WrappedDriver

How to install:

`pip install -e git+https://github.com/balexander85/wrapped_driver.git#egg=wrapped_driver`

```$python
from wrapped_driver import WrappedDriver

driver = WrappedDriver(
            chrome_driver_path=CHROME_DRIVER_PATH,
            browser="chrome",
            headless=headless,
         )
```
